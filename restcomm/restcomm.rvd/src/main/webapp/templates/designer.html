Call Control

Enabling CC will allow your application to get started over HTTP

Enable

Start action

Use this url to start this application using a simple GET request. A outgoing call will be made to the destination number specified and the RCML part of this application will run.

http://{{getRvdHost()}}:{{getRvdPort()}}/restcomm-rvd/services/apps/{{projectName}}/start

Call destination and origin

Add from and to paremeters to set the destination and originating numbers for the outgoing call.

http://{{getRvdHost()}}:{{getRvdPort()}}/restcomm-rvd/services/apps/{{projectName}}/start?from=12345&to=client:bob

Additional parameters

Additional parameters can be added to the above url. They will be available as application scoped RVD variables

http://{{getRvdHost()}}:{{getRvdPort()}}/restcomm-rvd/services/apps/{{projectName}}/start?from=12345&param1=foo&param2=bar
